# IronCliw Dashboard - Web Control Panel

A modern React-based web dashboard for IronCliw with real-time monitoring,
skill marketplace integration, conversation history, and system metrics.

## Features

- **Real-time Sub-Agent Monitoring**: Visual flow diagrams of agent execution
- **Skill Marketplace**: Browse, install, and rate 5400+ skills
- **Conversation History**: Searchable, filterable chat history
- **System Health**: Metrics, logs, and performance monitoring
- **Voice I/O Controls**: Voice input/output configuration
- **Model Hub**: Manage multiple AI providers and models

## Development

```bash
npm install
npm run dev
```
