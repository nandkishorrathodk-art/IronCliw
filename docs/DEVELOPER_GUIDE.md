# Developer Quick Start

## Setup Local Environment

1. Install Node.js v22+
2. Install pnpm: `npm install -g pnpm`
3. Clone repository
4. Run `pnpm install`

## Building

```bash
pnpm build
```

## Testing

```bash
pnpm test
```

## Debugging

Use the provided `.vscode/launch.json` to attach a debugger to the `Gateway` or `Daemon` processes.
