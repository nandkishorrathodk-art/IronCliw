# Burp Suite Repeater Tutorial

Repeater is used to manually modify and re-send individual HTTP requests.

### Steps to use Repeater:
1. Go to the **Proxy** tab (Shortcut: `Alt+P`).
2. Go to the **HTTP history** sub-tab.
3. Find the request you want to test.
4. Right-click the request and select **Send to Repeater** (Shortcut: `Ctrl+R`).
5. Switch to the **Repeater** tab (Shortcut: `Alt+R`).
6. Modify the parameters in the Request panel.
7. Click the **Send** button to fire the request.
8. Analyze the response in the Response panel.

### Important Shortcuts:
- Proxy Tab: `Alt+P`
- Repeater Tab: `Alt+R`
- Send to Repeater: `Ctrl+R`
- Issue Request: `Ctrl+G`
